# 🎮 Rock Paper Scissors Game (Python)

A fun and interactive **console-based Rock Paper Scissors game** built using Python.  
This project is beginner-friendly and demonstrates the use of core Python concepts like functions, loops, conditionals, and the `random` module.

---

## 📌 Features

- User vs Computer gameplay
- Multiple rounds
- Live scoreboard
- Input validation
- Emojis for better user experience 🎉
- Final winner announcement 🏆
- Clean and readable code structure

---

## 🛠 Technologies Used

- Python 3
- Built-in `random` module
